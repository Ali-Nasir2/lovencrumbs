# Love & Crumbs website

A responsive, dependency-free bakery website for **Love & Crumbs** in DFW, Texas.

## Preview

Open `index.html` in a browser, or run any local static server in this folder.

## Editing

Search the project for `[EDITABLE]` to find business details, prices, image slots, reviews, address, and form notes that should be replaced before launch.

- Replace each placeholder block with an `<img>` while retaining the surrounding class for sizing.
- Update the `hours` object in `script.js` for real opening hours.
- Connect the order and newsletter forms to your preferred form service before launch.
